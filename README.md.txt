A fun and educational game where you help Santa sort letters into categories! Test your speed and accuracy in this holiday-themed adventure.

## Features
- **Drag-and-Drop Gameplay**: Drag letters into the correct category bins.
- **3 Categories**: Toys, Books, and Sports.
- **Dynamic Gameplay**: Earn points, beat the timer, and aim for the high score!
- **Festive Background Music**: Enjoy a Christmas-themed soundtrack.


## Installation
1. Install Python 3.10+ on your computer.
2. Install Pygame by running:
pip install pygame


3. Download or clone this repository.

## How to Run
1. Open a terminal and navigate to the game folder.
2. Run the following command:
python mygame1.py

3. Enjoy the game!



## Controls
- **Mouse**: Drag letters into the correct bins.
- **Spacebar**: Start or restart the game.
